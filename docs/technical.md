# مستند فنی

## معماری سیستم
- معماری سه‌لایه (کاربر، منطق برنامه، پایگاه داده)
- استفاده از API برای ارتباط بین بخش‌ها

## زبان‌ها و ابزارها
- Frontend: HTML, CSS, JavaScript (یا React)
- Backend: Python (Django/Flask) یا Node.js
- Database: MySQL/PostgreSQL

## ساختار پوشه‌ها
- docs/ → مستندات
- src/ → کد اصلی
- tests/ → تست‌ها
- reports/ → گزارش‌ها

## وابستگی‌ها
- کتابخانه‌های احراز هویت
- کتابخانه‌های مدیریت پایگاه داده
