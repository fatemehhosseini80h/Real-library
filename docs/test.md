# مستند تست

## استراتژی تست
- تست واحد برای ماژول‌های ثبت‌نام، جستجو، رزرو
- تست یکپارچه برای کل فرآیند امانت کتاب
- تست کارایی برای تعداد زیاد کاربران همزمان

## ابزارهای تست
- PyTest یا Jest
- ابزار شبیه‌سازی بار (Load Testing)

## سناریوهای تست
- کاربر جدید ثبت‌نام می‌کند → باید موفق شود
- کاربر کتابی را رزرو می‌کند → باید در لیست رزرو ثبت شود
- کاربر کتاب را دیر بازمی‌گرداند → باید هشدار دریافت کند

## نتایج تست
- جدول نتایج تست‌ها
- مشکلات کشف‌شده و راه‌حل‌ها
