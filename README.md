# توضیح کلی پروژه 
این پروژه یک سامانه امانت‌دهی کتاب آنلاین است که به کاربران این امکان را می‌دهد تا کتاب‌های خوانده شده برای امانت دهی را جستجو کرده، بررسی کنند و درخواست امانت دهند.
هدف این سامانه، ایجاد فضایی ساده و کاربرپسند برای دسترسی به کتاب‌ها و مدیریت درخواست‌های امانت در یک بستر آنلاین است.

# ویژگی‌ها
ثبت و مدیریت کتاب‌ها: امکان ثبت کتاب‌های جدید، ویرایش اطلاعات کتاب‌ها و حذف آن‌ها.

جستجو و فیلتر کتاب‌ها: کاربران می‌توانند کتاب‌ها را بر اساس عنوان، نویسنده، ژانر و یا سال انتشار جستجو کنند.

مدیریت درخواست‌های امانت: کاربران می‌توانند درخواست امانت کتاب کنند و وضعیت درخواست خود را پیگیری کنند.

مدیریت کاربران: امکان ثبت‌نام و ورود کاربران به سیستم برای دسترسی به امکانات.

تاریخچه امانت‌ها: ثبت تاریخچه امانت‌ها و بازگشت کتاب‌ها.
